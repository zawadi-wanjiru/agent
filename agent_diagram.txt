Agent Roles:

1. Greeter: "What does the customer need?"
   - Tools: none
   - Job: classify intent

2. Researcher: "Find the answer in our docs"
   - Tools: search_faq
   - Job: look up policies, procedures

3. Order Specialist: "Get order details"
   - Tools: lookup_order
   - Job: track shipments, check status

4. Resolver: "What action should we take?"
   - Tools: log_action
   - Job: plan refunds, escalations, tickets

5. Quality Reviewer: "Is this answer good enough?"
   - Tools: none
   - Job: critique response, ensure accuracy

6. Supervisor: "Who needs to work on this?"
   - Tools: none
   - Job: coordinate team, decide workflow
   - Note: Currently using routing logic instead for speed
